# Vámosi Gergő — Portfólió

Egyszerű, statikus HTML + CSS + JavaScript portfólió, GitHub Pages-re előkészítve.

## Személyre szabás

1. Cseréld le az `assets/profile-placeholder.svg` fájlt a saját képedre, és az `index.html` fájlban írd át a kép `src` értékét.
2. A PDF-eket helyezd az `assets/documents` mappába; a pontos névlista az ottani README-ben van.
3. Cseréld a Lorem Ipsum szövegeket saját tartalomra, valamint a `hello@example.com` e-mail címeket a sajátodra.

## GitHub Pages publikálás

1. Hozz létre egy GitHub repository-t, majd töltsd fel ennek a mappának a tartalmát a repository gyökerébe.
2. GitHubon nyisd meg: **Settings → Pages**.
3. A **Deploy from a branch** opciónál válaszd a `main` ágat és a `/(root)` mappát, majd mentsd el.

A weboldal pár percen belül elérhető lesz a GitHub által megadott címen.
